# Neural Networks Basics

## Forward pass
A feed-forward network computes layer activations from input to output.
Nonlinear activation functions let stacked layers represent complex functions.

## Backpropagation
Backpropagation applies the chain rule to compute gradients of the loss with respect to each parameter.
These gradients are used by optimization methods such as gradient descent.

## Overfitting and generalization
A model can overfit by memorizing training data patterns that do not transfer.
Validation performance is used to monitor generalization.
